## Joseph Dattilo

Developer. Technical founder. Building AI-native tooling in public.

I ship production software with AI agents as first-class contributors — not demos, not experiments. I think agentic development is the most significant shift in how software gets built since version control itself. Most of what I'm building right now is designed to make that safer and more reliable.

Most of my code is still private, but that's changing.

### What I'm building

**[RepoHarbor](https://repoharbor.com)** — a self-hosted git control plane for teams running AI coding agents. Enforces branch protection, requires pull requests, keeps credentials server-side. Because agents forget branches exist. (I watched mine do it while I was in the room.)
